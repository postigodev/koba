# Koba Brand Assets

This directory contains the core Koba PNG brand assets.

- `koba-logo-horizontal.png`: preferred logo for README files, project pages, and wide header placements.
- `koba-mark.png`: standalone mark for icons, favicons, and compact identity uses.
- `koba-wordmark.png`: wordmark for headers or layouts where the mark is already present elsewhere.
- `koba-logo-stacked.png`: stacked logo for square-ish placements and tighter vertical layouts.
